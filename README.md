# Writeups

An intentionally public-facing repo where I publish:
- polished writeups of various CTFs and other challenges
- breakdowns of certain topics of interest to me
- project summaries for stuff I build

...all with the goal of improving and documenting my own learning.

- **I will never publish flags here.**
- **I will never publish information related to time-bound events until after the deadline has been reached and the event is officially closed.**

## Structure
- `breakdowns` - semi-deep dives on a specific topic
- `projects` - things I build
- `writeups` - home base for CTFs and cybersec challenges
    - `pico-ctfs` - picoGym and other challenges
    - `THM` - public challenges from TryHackMe

Each writeup includes:
- A description of the challenge
- My step-by-step process
- Relevant code snippets
- Screenshots or command output
- Takeaways / lessons learned
